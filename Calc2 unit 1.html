Calculus II Study Guide – Chapters 2 and 3

2.0A – Basic Integration Rules

General Rule
[ \int f(x)\,dx = F(x) + C ]

Power Rule
[ \int x^n dx = \frac{x^{n+1}}{n+1} + C \quad \text{for } n \ne -1 ]

Logarithmic
[ \int \frac{1}{x} dx = \ln|x| + C ]

Exponential
[ \int e^x dx = e^x + C ] [ \int a^x dx = \frac{a^x}{\ln a} + C \quad (a > 0,\ a \ne 1) ]

Trigonometric
[ \int \sin x\,dx = -\cos x + C ] [ \int \cos x\,dx = \sin x + C ] [ \int \sec^2 x\,dx = \tan x + C ] [ \int \csc^2 x\,dx = -\cot x + C ] [ \int \sec x \tan x\,dx = \sec x + C ] [ \int \csc x \cot x\,dx = -\csc x + C ]

2.0B – Substitution

Formula
[ \int f(g(x))g'(x)\,dx = \int f(u)\,du ]

Definite Integral Version
[ \inta^b f(g(x))g'(x)\,dx = \int{u(a)}^{u(b)} f(u)\,du ]

2.1 – Areas Between Curves

Vertical Slicing
[ \int_a^b [f(x) - g(x)]\,dx ]

Horizontal Slicing
[ \int_c^d [f(y) - g(y)]\,dy ]

2.2 – Volume by Slicing (Washer Method)

Horizontal Rotation
[ V = \pi \int_a^b \left( [R(x)]^2 - [r(x)]^2 \right) dx ]

Vertical Rotation
[ V = \pi \int_c^d \left( [R(y)]^2 - [r(y)]^2 \right) dy ]

2.3 – Cylindrical Shells

Vertical Axis of Rotation
[ V = 2\pi \int_a^b x(\text{height})\,dx ]

Horizontal Axis of Rotation
[ V = 2\pi \int_c^d y(\text{width})\,dy ]

3.1 – Integration by Parts

Formula
[ \int u\,dv = uv - \int v\,du ]

3.2 – Trigonometric Integrals

Identities [ \sin^2 x = \frac{1 - \cos(2x)}{2} ] [ \cos^2 x = \frac{1 + \cos(2x)}{2} ] [ \sin(2x) = 2\sin x \cos x ] [ \cos(2x) = \cos^2 x - \sin^2 x ]

3.3 – Trigonometric Substitution

Patterns [ \sqrt{a^2 - x^2} \Rightarrow x = a\sin\theta ] [ \sqrt{a^2 + x^2} \Rightarrow x = a\tan\theta ] [ \sqrt{x^2 - a^2} \Rightarrow x = a\sec\theta ]

3.4 – Partial Fractions

Form [ \frac{P(x)}{Q(x)} \quad \text{where } \deg(P) < \deg(Q) ]

Decomposition Types - Linear: ( \frac{A}{x - r} ) - Repeated Linear: ( \frac{A}{x - r} + \frac{B}{(x - r)^2} + \dots ) - Irreducible Quadratic: ( \frac{Ax + B}{x^2 + bx + c} )

3.6 – Numerical Integration

Trapezoidal Rule
[ \frac{b - a}{2n} [f(x0) + 2f(x1) + \dots + 2f(x{n-1}) + f(xn)] ]

Simpson’s Rule
[ \frac{b - a}{3n} [f(x0) + 4f(x1) + 2f(x2) + \dots + 4f(x{n-1}) + f(x_n)] ]

3.7 – Improper Integrals

Infinite Limits [ \inta^\infty f(x)\,dx = \lim{t \to \infty} \int_a^t f(x)\,dx ]

Vertical Asymptotes [ \inta^b f(x)\,dx = \lim{t \to c^-} \int_a^t f(x)\,dx \quad \text{if } f(x) \to \infty \text{ at } x = c ]

3.8 – L’Hôpital’s Rule

Forms
- ( \frac{0}{0} ) - ( \frac{\infty}{\infty} )

Rule [ \lim{x \to a} \frac{f(x)}{g(x)} = \lim{x \to a} \frac{f'(x)}{g'(x)} \quad \text{if the limit exists} ]
