Trigonometric Substitution
If you see \( \sqrt{a^2 - x^2} \), set \( x = a\sin\theta,\quad dx = a\cos\theta\,d\theta \).
If you see \( \sqrt{a^2 + x^2} \), set \( x = a\tan\theta,\quad dx = a\sec^2\theta\,d\theta \).
If you see \( \sqrt{x^2 - a^2} \), set \( x = a\sec\theta,\quad dx = a\sec\theta\tan\theta\,d\theta \).
