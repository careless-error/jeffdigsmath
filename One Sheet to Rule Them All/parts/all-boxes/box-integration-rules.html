Integration
Power Rule \( \int x^n\,dx \) = \( \frac{x^{n+1}}{n+1} + C \)
Log Rule \( \int \frac{1}{x}\,dx \) = \( \ln |x| + C \)
Exponential (\(e^x\)) \( \int e^x\,dx \) = \( e^x + C \)
Exponential (\(a^x\)) \( \int a^x\,dx \) = \( \frac{a^x}{\ln a} + C \)
Sine \( \int \sin x\,dx \) = \( -\cos x + C \)
Cosine \( \int \cos x\,dx \) = \( \sin x + C \)
Secant Squared \( \int \sec^2 x\,dx \) = \( \tan x + C \)
Cosecant Squared \( \int \csc^2 x\,dx \) = \( -\cot x + C \)
Sec × Tan \( \int \sec x \tan x\,dx \) = \( \sec x + C \)
Csc × Cot \( \int \csc x \cot x\,dx \) = \( -\csc x + C \)
