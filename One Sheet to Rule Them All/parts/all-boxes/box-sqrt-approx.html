Square Root Approximation
Use tangent line to estimate \( \sqrt{x} \) near a known value
If \( f(x) = \sqrt{x} \), then \( f'(x) = \dfrac{1}{2\sqrt{x}} \)
Linear approx: \( f(x + \Delta x) \approx f(x) + f'(x)\Delta x \)
Example: \( \sqrt{10} \approx \sqrt{9} + \dfrac{1}{2\sqrt{9}}(1) = 3 + \dfrac{1}{6} = 3.166\ldots \)
